nfcpy sample (suica reader)
======================

m2wasabiさんのサンプルをpython3対応にしています。

## About  
nfcpy 0.10.0用の Felica読み込みサンプル

## Using  
  + python
  + nfcpy

## Setup  
1. Install nfcpy

```
cd python
bzr branch lp:nfcpy
chmod a+x *.py
cd ..
```

## Contents  
+ tagspec.py
  読み込んだNFCタグのクラス・メソッドを表示
+ nfc_dump_suica.py
  読み込んだSuidaの生データを表示
+ suica_read.py
  読み込んだSuidaの分析データを表示

それぞれコマンドラインから実行します。
